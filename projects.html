Puffy Sheep Burger 點餐系統

從 Python 邏輯到網頁互動的完整實作

專案簡介

本專案初衷是將資管系課堂所學的 Python GUI 邏輯，轉化為更易於分享與展示的 Web 介面。透過 HTML5, Bootstrap 5 與 JavaScript，成功復刻了原本在電腦端運行的點餐與折扣邏輯。

關鍵技術點：
  • JavaScript 運算：即時計算多個品項金額，並自動判定滿 $500 享 9 折。
  • Modal 互動：採用全螢幕彈窗介面，優化行動裝置使用者的操作體驗。
  • 視覺優化：結合 AOS 動畫與自訂 CSS，打造現代化的 UI 感官。
